# 이어봄 - 제품 문서

## 서비스 개요

**이어봄**은 언어치료사와 학부모가 치료 일정을 실시간으로 공유하고, 일정 변경과 취소를 즉시 확인할 수 있는 치료 스케줄 공유 서비스다.

## 핵심 문제

치료사와 학부모 사이에서 일정 공유가 전화/문자에 의존하고 있어:

- 일정 변경 시 개별 연락이 필요해 치료사의 시간이 소모된다.
- 학부모가 변경 사항을 놓치거나 착오가 발생한다.
- 일정 확인 여부를 양쪽 모두 확인할 수 없다.

## 핵심 사용자

### 치료사

- 언어치료사
- 여러 아동을 동시에 관리
- 일정 등록, 변경, 취소 주체

### 학부모

- 자녀의 치료 일정 확인자
- 변경된 일정 확인 처리
- 알림 수신

## MVP 핵심 흐름

```
1. 치료사 회원가입/로그인
2. 치료사가 아동 등록
3. 치료사가 초대 코드 생성
4. 학부모 회원가입/로그인
5. 학부모가 초대 코드 입력 → 아동 연결
6. 치료사가 치료 일정 등록/수정/취소
7. 학부모가 아동의 치료 일정 조회
8. 학부모가 변경된 일정 확인 처리
9. 일정 변경 시 학부모에게 알림 생성
10. 치료사가 반복 일정 생성 (주간/격주)
```

## 핵심 지표 (KPI)

- 일정 확인율: 학부모가 변경 알림 후 24시간 내 확인하는 비율
- 주간 활성 치료사 수
- 치료사당 연결된 학부모 수

## 사용자 스토리 (Phase 별)

### Phase 2: 인증

- 치료사로 회원가입할 수 있다.
- 학부모로 회원가입할 수 있다.
- 이메일/비밀번호로 로그인할 수 있다.
- JWT 토큰으로 인증 상태를 유지할 수 있다.

### Phase 3: 핵심 기능

- 치료사가 아동을 등록할 수 있다.
- 치료사가 초대 코드를 생성할 수 있다.
- 학부모가 초대 코드로 아동과 연결될 수 있다.
- 치료사가 일정을 등록/수정/취소할 수 있다.
- 학부모가 아동의 일정을 조회할 수 있다.
- 일정 변경 시 학부모에게 알림이 생성된다.
- 학부모가 일정을 확인 처리할 수 있다.

### Phase 4: 고도화

- 반복 일정을 생성할 수 있다.
- 푸시 알림을 받을 수 있다.
- 치료 기록을 남길 수 있다.
